# 🌐 Translator Pro (Azure Translator API)

A modern, web-based translation tool inspired by Google Translate, powered by **Microsoft Azure Translator API**.
This project allows users to enter their own Azure API credentials and translate text in real-time with a clean and professional UI.

---

## 🚀 Features

* 🔑 User-provided Azure API Key, Endpoint, and Region
* 🌍 Multi-language translation support
* 🔄 Swap input/output text
* 📋 Copy translated text
* 🔢 Live character count
* 💻 Clean and responsive UI
* ⚡ Fast real-time translation using Azure AI

---

---

## 🔑 Azure Setup

To use this app, you need Azure Translator credentials:

1. Go to Azure Portal

2. Create a **Translator Resource**

3. Copy:

   * API Key
   * Endpoint
   * Region

4. Paste them into the app input fields

---

## ▶️ Usage

1. Enter your Azure credentials
2. Type or paste text in the input box
3. Select target language
4. Click **Translate**
5. View output instantly

---

## ⚠️ Important Note

This app runs entirely on the frontend, so:

👉 Your API key is visible in the browser

For production use:

* Use a backend (Node.js / Python)
* Store API keys securely
